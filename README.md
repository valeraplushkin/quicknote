# QuickNote

Мгновенный захват мыслей — десктопное приложение для Windows 
с глобальными хоткеями.

## Возможности
- Ctrl+N — мгновенный захват заметки
- Ctrl+Shift+N — захват с автозакреплением
- Ctrl+Alt+N — список всех заметок
- Ctrl+L — закрепить текущую заметку
- Минималистичный, нативный интерфейс без задержек

## Стек
- Tauri 2.0 (Rust)
- React + TypeScript
- Vite
- SQLite (sqlx)
